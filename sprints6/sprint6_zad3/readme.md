 # Tower Defense Game

## Описание

Tower Defense Game — это увлекательная стратегия, в которой игрокам необходимо защищать свою базу, размещая башни и уничтожая волны врагов. Игроки могут создавать различные типы башен и улучшать их, чтобы остановить наступление врагов на своем пути.

# Tower Defense Game

## Описание проекта

Tower Defense Game - это игра в жанре "Защита башни", в которой игрок должен защищать свою базу от волн врагов, расставляя различные типы башен. Игра предлагает стратегический элемент, позволяя игрокам выбирать и размещать башни, каждую с уникальными характеристиками.

## Основные функции

- Игрок может расставлять башни для защиты от волн врагов.
- Поддержка реализации различных типов башен с уникальными характеристиками (например, базовая башня и снайперская).
- Враги движутся по заранее заданному пути, и игрок должен предотвратить их достижение базы.
- Система улучшения башен, которая позволяет игроку увеличивать мощность башен за деньги, полученные в игре.
- Звук выстрелов и появления врагов для улучшенного игрового опыта.

### Описание классов

- **Tower**: Базовый класс для всех типов башен, управляющий логикой стрельбы и поиска целей.
- **BasicTower**: Класс для создания базовой башни с определёнными характеристиками.
- **SniperTower**: Класс для снайперской башни, которая имеет большую дальность стрельбы и наносит больше урона.
- **Enemy**: Класс для врагов, которые движутся по заданному пути и могут получать урон.
- **Bullet**: Класс для снаряда, который управляет движением пули и проверяет попадания в врагов.
