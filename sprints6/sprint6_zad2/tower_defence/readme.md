 # Tower Defense Game

## Описание

Tower Defense Game — это увлекательная стратегия, в которой игрокам необходимо защищать свою базу, размещая башни и уничтожая волны врагов. Игроки могут создавать различные типы башен и улучшать их, чтобы остановить наступление врагов на своем пути.

## Установка

### Необходимые зависимости

Для запуска игры вам потребуется Python и необходимые библиотеки. Убедитесь, что у вас установлены:

- **Python 3.x**
- **Pygame**

### Установка библиотеки Pygame

Вы можете установить библиотеку Pygame, используя pip:

```bash
pip install pygame
